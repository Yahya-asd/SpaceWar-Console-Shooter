//////       USE THE PROVIDED 'help.h' FILE AS SOME MODIFICATIONS WERE MADE TO BE ABLE TO USE MORE KEYS     ///////


1)   The main game mode is the hard mode which has all the required features 
The rest of the features are also present in the other modes however the save and load is only available in the hard mode
 

2)   To run the game simply add the provided header file in your new project then either copy paste the code from the provided file
or just add the "c++" file in your project.


3) Run the game using F5 or start code debugging .


4) The instructions are as follows :
ESC  -       To pause and resume the game
UP ARROW -   To move the ship up
DOWN ARROW - To move the ship down
RIGHT ARROW -To move the ship right
LEFT ARROW - To move the ship left
SPACE BAR  - To Fire a bullet
SHIFT   -    To save the current state
L   -        To load the previous state
